## История изменений

- 0.3.0
  - Спринт 12.
  - Добавлена ER-диаграмма будущей базы данных
- 0.2.0
  - Спринт 11.
  - Добавлены сервисы и репозитории.
  - Организовано хранение данных в оперативной памяти.

  - Спринт 10. 
  - Первая версия приложения.
  - Вся реализация находится в контроллерах.
  - Тесты
